# Сервис платного отчёта «AI-видимость» — как запустить

Платный продукт: клиент вводит сайт на лендинге, оплачивает 1290 ₽ через ТBank,
сервис опрашивает 7 нейросетей, собирает PDF-отчёт и присылает его клиенту в Telegram.

Бесплатный технический аудит остаётся на Cloudflare Worker и не меняется.

## Что в папке

```
app.py            сервис: приём заявки, оплата ТBank, вебхуки, доставка в Telegram
engine.py         движок: генерация запросов, опрос 7 нейросетей, детекция упоминаний
build_report.py   сборка брендированного PDF из данных движка
Gilroy-*.ttf      4 шрифта Gilroy (встраиваются в PDF)
avatar.png, cover-bg.jpg, qr_tg.png   фото в круге, обложка-горы, QR на канал
requirements.txt  зависимости
Dockerfile        для деплоя на любой контейнерный хостинг
```

## Как это работает (поток)

1. На лендинге клиент вводит сайт (и нишу) и нажимает «Получить отчёт».
2. Сайт шлёт запрос на `/create-payment`, сервис создаёт заказ и платёж ТBank, возвращает ссылку на оплату.
3. Клиент платит. ТBank подтверждает оплату вебхуком `/tbank/notify` (заказ помечается оплаченным).
4. Клиента перебрасывает в Telegram-бот, он жмёт «Старт». Сервер проверяет, что заказ оплачен, и запускает движок: 10 запросов × 7 нейросетей × 2 прогона, ищет упоминания бренда и конкурентов, считает видимость.
5. `build_report.py` собирает PDF в твоём стиле.
6. Бот присылает клиенту готовый PDF прямо в чат.

## Что нужно от тебя (один раз)

1. **Хостинг** для сервиса (Python нельзя на reg.ru, нужен контейнерный хост). Бесплатные варианты: Render, Railway, Fly. Нужен постоянный диск для папки `/data`.
2. **Пароль терминала ТBank** (демо: `1782125233968DEMO`). Впишешь сам в секреты хостинга, мне его присылать не нужно.
3. **Telegram-бот**: создать через @BotFather, получить токен.
4. **API-ключи 7 нейросетей** (по мере появления, без них сервис работает в мок-режиме).

## Секреты (переменные окружения)

| Переменная | Зачем | Обязательна |
|---|---|---|
| `TBANK_TERMINAL` | терминал (демо `1782125233968DEMO`) | да |
| `TBANK_PASSWORD` | пароль терминала | да |
| `BASE_URL` | адрес сервиса, напр. `https://service.annakurbatova.ru` | да |
| `PRICE_RUB` | цена, по умолчанию 1290 | нет |
| `TELEGRAM_BOT_TOKEN` | токен бота от @BotFather | да |
| `TELEGRAM_BOT_USERNAME` | имя бота без @ | да |
| `TEST_MODE` | `1` = мок-движок без ключей (для проверки) | нет |
| ключи нейросетей | см. таблицу ниже | по мере |
| `SMTP_HOST/PORT/USER/PASS`, `MAIL_FROM` | резервная отправка на почту | нет |

## API-ключи нейросетей

| Движок | Переменная | Доступ |
|---|---|---|
| Яндекс Нейро (YandexGPT) | `YANDEX_API_KEY` + `YANDEX_FOLDER_ID` | РФ, без зарубежной карты |
| GigaChat (Сбер) | `GIGACHAT_API_KEY` | РФ, без зарубежной карты |
| DeepSeek | `DEEPSEEK_API_KEY` | зарубежный, ключ достать реально |
| Perplexity | `PERPLEXITY_API_KEY` | зарубежный аккаунт и карта |
| ChatGPT (OpenAI) | `OPENAI_API_KEY` | зарубежный аккаунт и карта |
| Claude (Anthropic) | `ANTHROPIC_API_KEY` | зарубежный аккаунт и карта |
| Gemini (Google) | `GEMINI_API_KEY` | зарубежный аккаунт |

Сервер до зарубежных API достучится (он не в РФ), нужны только твои ключи.
Пока ключа нет, движок по этому движку отдаёт мок-данные, и это видно в `/health`.

## Деплой (рекомендую Railway)

Railway не «засыпает» (важно для вебхуков и фоновой генерации), деплоит из GitHub через UI,
стартовый кредит $5 без карты, дальше Hobby ~$5/мес.

1. Залей файлы сервиса в репозиторий на GitHub (можно через веб: New repository → Upload files).
2. railway.com → войти через GitHub → New Project → Deploy from GitHub repo → выбери репозиторий.
   Railway сам увидит `Dockerfile` и соберёт образ.
3. Variables → впиши секреты из таблицы выше (для начала добавь `TEST_MODE=1`, чтобы проверить без ключей).
4. Settings → Networking → Generate Domain → получишь адрес вида `xxx.up.railway.app`.
   Впиши его в переменную `BASE_URL`.
5. Открой `https://<домен>/health` — увидишь статус терминала, бота и ключей.
6. Постоянное хранение (необязательно): добавь Volume на путь `/data` и задай
   `DB_PATH=/data/orders.db`, `REPORTS_DIR=/data/reports`.

Альтернатива: Render. Есть бесплатный тариф, но веб-сервис засыпает через 15 минут простоя
(старт 30-60 сек), для боевого режима лучше платный Starter $7/мес.

## Подключить Telegram-бота

1. @BotFather → `/newbot` → получишь токен → впиши в `TELEGRAM_BOT_TOKEN`, имя в `TELEGRAM_BOT_USERNAME`.
2. Привяжи вебхук (один раз, подставь свои значения):
   `https://api.telegram.org/bot<ТОКЕН>/setWebhook?url=<BASE_URL>/telegram/webhook`

## Подключить оплату ТBank

1. В кабинете ТBank возьми пароль демо-терминала, впиши в `TBANK_PASSWORD`.
2. NotificationURL терминала: `<BASE_URL>/tbank/notify` (сервис ставит автоматически в платеже).
3. Демо-терминал позволяет обкатать весь поток без реальных денег.

## Проверка без ключей и оплаты

Локально: `TEST_MODE=1 python3 engine.py` соберёт данные, затем
`python3 build_report.py --data report_data.json --out test.pdf` соберёт PDF.
Так можно увидеть итоговый отчёт на демо-данных.

## Связать с сайтом

Кнопка «Получить полный отчёт» ведёт на форму (сайт + ниша), форма шлёт POST на
`<BASE_URL>/create-payment` и перенаправляет клиента на ссылку оплаты из ответа.
